# 💍 WeddingSalt2

**WeddingSalt2** es una versión alternativa del sitio web de bodas, desarrollada con React. Este proyecto se centra en proporcionar una experiencia única y personalizada para eventos especiales.

## ✨ Características

- Diseño moderno y responsivo.
- Secciones dedicadas a paquetes y servicios.
- Portafolio visual de eventos anteriores.
- Blog con artículos y novedades.
- Formulario de contacto para consultas.
- Integración con WhatsApp para comunicación directa.

## 🛠️ Tecnologías Utilizadas

- [React](https://reactjs.org/)
- [React Router](https://reactrouter.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Icons](https://react-icons.github.io/react-icons/)

## 📂 Estructura del Proyecto

- `src/components`: Componentes reutilizables como Navbar y Footer.
- `src/pages`: Páginas principales del sitio web.
- `src/pages/servicios`: Detalles de servicios ofrecidos.
